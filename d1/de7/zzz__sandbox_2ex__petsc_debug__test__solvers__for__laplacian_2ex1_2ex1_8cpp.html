◆ AssemblePoissonProblem_AD()

void AssemblePoissonProblem_AD ( MultiLevelProblem &  ml_prob)

This function assemble the stiffnes matrix KK and the residual vector Res Using automatic differentiation for Newton iterative scheme J(u0) w = - F(u0) , with u = u0 + w

  • F = f(x) - J u = Res J = \grad_u F

thus J w = f(x) - J u0
