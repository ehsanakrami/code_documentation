Detailed Description

This class is an abstract class for input parsing. It implements the Factory Method pattern. Using the static build function you can istantiate the concrete classes HTMLInputParser (not yet implemented) or JsonInputParser.

How to use it: the path of the variable in the json input file is expressed as a sequence of words separated by a point e.g. : "mesh.first.type.box"

use the getValue function to read and get the value use the getSize function to get the size of a vector use the isTrue function to check if the basepath.type value in the file is equal or not to the type value you ask to
