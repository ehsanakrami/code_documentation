Detailed Description

This example shows how to set and solve the weak form of the Boussinesq appoximation of the Navier-Stokes Equation.

in a unit box domain (in 2D and 3D) with given temperature 0 and 1 on the left and right walls, respectively, and insulated walls elsewhere.
